operador this - sempre vai fazer referência ao parâmetro da função

'this' é o operador que indica que a variável criada dentro da função, vai receber um parâmetro da função

serve para distinguir qual é a variável e qual é o parÂmetro, sem ter que declarar um var let ou const


em outras palavras:

podemos criar uma 'variável: nome' e falar que ela recebe o parâmetro 'nome':


quando queremos trabalhar com o valor do parâmetro dentro de uma variável, dentro da função!!!


function aluno(nome, nota){
let nome = nome
}

'operador this' vai indicar que a variável criada dentro da função, faz referência ao elemento(ao parâmetro) da 'função aluno'


function aluno(nome, nota){
this.nome = nome this.nota = nota
}
