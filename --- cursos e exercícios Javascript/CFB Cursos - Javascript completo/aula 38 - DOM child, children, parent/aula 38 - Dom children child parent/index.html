Entendendo a relação dos elementos no DOM em Javascript

Método, Propriedade, filhos, irmãos, Pai, avó do DOM

caixinha azul - é um PROPRIEDADE. Então NÃO COLOCA o parênteses para ser executado
caixinha vermelha - é um MÉTODO (). Então PRECISA de um parênteses para ser executado

child - filho


childNode (nó) - os textos dentro da div são considerados como child, e contabilizam também.
firstChild - retorna o primeiro elemento dentro do pedido, e se esse elemento tiver um texto, ele retorna entao o texto
firstChild retorna as tags e os elementos dentro das tags
lastChild - retorna o último elemento dentro do pedido, e se esse elemento tiver um texto, ele retorna entao o texto
lastChild retorna as tags e os elementos dentro das tags

children - retorna somente as tags. Retorna os elementos filhos da div, sem contar os textos
firstElementChild - retorna apenas as tags (os elementos). Sem retornar o texto
lastElementChild - retorna apenas as tags do último elemento dentro do pedido. Sem texto
