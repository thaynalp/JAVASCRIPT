parseFloat()

var valorStr = '50.5'; var parseValorStr = parseFloat(valorStr); typeof valorStr // string typeof parseValorStr // number

A função parseFloat analisa um argumento (convertendo-o para uma string primeiro caso necessário) e retorna um número de ponto flutuante (número decimal).

parseFloat(num) converte num em um número de ponto flutuante a partir da string. 1 - transformamos o número em decimal 2 - nesse processo ele vai perder o sinal de menos e os zeros 3 - para recuperar o sinal negativo, original, vamos usar o *math.sign()
