crie uma função que inverta um número

Passo a passo:
*obs: o parseFloat vem primeiro
1- tranformar de nº para string 'toString()' - para poder usar o split
2- converter a string criada em um array - 'split('')' - para acessar elemento por elemento
3 - inverter a ordem dos números do array - 'reverse()'
4 - unir o array novamente, e transformá-lo numa string 'join('')'
5 - converter os nºs inteiros em decimais 'parseFloat() -(vai cortar os zeros e o sinal)
6 - multiplicar por '*Math.sign()' - devolve o sinal original perdido no processo anterior
