Enviar dados pro local storage

localStorage.setItem('chave','value')

Nesse exercício a chave é 'db_client' e o valor é 'client' 'client' é um callback que vem da base de dados temporária chamada 'tempClient', em formato JSON.

O localStorage só aceita dados em formato STRING. vamos então usar o método JSON.stringify

JSON.stringify()

é um método estático converte um valor JavaScript em uma string JSON
    Sintaxe:
  • JSON.stringify(value)
  • JSON.stringify(value, replacer)
  • JSON.stringify(value, replacer, space)
