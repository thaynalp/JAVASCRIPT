Passo 3: Enviar dados pro local storage.

Fluxo do código:

  1. criar uma função separada para levar e trazer os dados pro localStorage
  2. chamar essas funções de levar e trazer dados dentro da função createClient
  3. getLocalStorage vai pegar os dados no localStorage e retornar para a variável 'dbClient', que está dentro da função createClient
  4. a variável 'dbClient', que está recebendo um array, vai enviar dados do 'client' pelo 'push'. Esses dados estão vindo direto do 'objeto' 'tempClient'
  5. a função setLocalStorage vai receber por parâmetro 'dbClient'. O valor lançado para o localStorage será o valor recebido por 'dbClient'
  6. e a função 'createClient' vai receber o parâmetro 'client', que são dados que esão vindo do objeto 'tempClient'
  7. e essa função vai receber a função 'setLocalStorage', que vai enviar dados do 'dbClient'
obs: 'dbClient' é uma variável que está recebendo dados do array que está no localStorage e está sendo enviado dados também, pela função 'setLocalStorage(dbClient)
