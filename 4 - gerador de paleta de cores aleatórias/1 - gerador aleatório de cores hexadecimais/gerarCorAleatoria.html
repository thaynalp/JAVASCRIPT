COMO GERAR CORES ALEATÓRIAS NO JAVASCRIPT?

código para gerar cores hexadecimais aleatórias
'#' + Math.floor(Math.random() * 0x1000000).toString(16).padStart(6, '0');
let str = '#';
while (str.length < 7) { str +=Math.floor(Math.random() * 0x10).toString(16); }
Breve explicação da expressão acima:
1 - A função Math.floor arredonda o número para baixo, de modo a remover a parte decimal do número gerado pela função Math.random.
2 - O método Number.prototype.toString(16) converte o número em string em sua representação hexadecimal.
3 - O método String.prototype.padStart(6, '0') garante que a string hexadecimal tenha sempre seis dígitos.
